For viewing in uGo-Man (animated comic) format

PC

Open index.html directly in Chrome or another browser.


Android

  1. Open index.html from a file manager app (e.g., Files by Google) in your browser.
  2. Tap the “Select Folder” button and choose the folder containing index.html.
  3. Allow access and confirm “Upload” to start viewing.
    Note: Although “Upload” is displayed, it is processed locally and no network connection is required.

Tip: Bookmark the displayed page to reduce steps for future access.


iOS

Method 1 (Recommended)

  1. Open index.html in a file management app.
  2. Tap the “Copy Bookmarklet” button.
  3. Open Safari and bookmark any page.
  4. Edit the bookmark.
    On iOS 26: Long-press it, then swipe up from the bottom to reveal the “Edit” button.
    Delete the existing URL and paste the long code into the URL field. You may enter any name you like.
  5. When you open that bookmark, the “Select Folder” button will appear.
    Choose the folder that contains index.html and the viewer will open in Safari.
  6. From then on, you can access it via the Safari bookmark and select the folder to view.

    7. (※ This will not work from a blank tab. Please open any page first before using the bookmarklet.)

Method 2

  1. Open index.html in an HTML viewer or a file management app.
  2. Tap the “Select Folder” button and choose the folder that contains index.html.
    ※ Confirmed working on HTML Viewer Q (Spica) / iOS 26 / iPhone SE2.
    ※ In some apps (e.g., Documents by Readdle), tapping a video may force fullscreen playback.
